Ganancias

La aplicacion de Ganancias esta hecha para Android y fue construida a la medida. El lenguaje de programación utilizado es Java y tiene una base de datos en SQLite

Las funcionalidades son las siguientes:

Registra tus ingresos, gastos e inversion cada que lo necesites

Verifica que llenes correctamente tu formulario

Los iconos de los registros cambian indicando el estatus de tu registro de esta forma:

  • Ingresos > Gastos ➡ Icono verde
  • Ingresos = Gastos ➡ Icono amarillo
  • Ingresos < gastos ➡ Icono rojo

Recibe notificaciones cada quincena

Abre la notificación y consulta un resumen de tu quincena

Consulta los registros de cada quincena

Exporta tus registros a pdf con el icono que esta en la parte superior derecha

El pdf se encuentra dentro de tus archivos, en la carpeta Ganancias
